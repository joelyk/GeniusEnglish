# GeniusEnglish

Mini site d'entraînement en anglais avec modules de lecture + QCM interactifs.
Tout est en HTML/CSS/JS (sans backend), idéal pour GitHub Pages.

## Auteur
Mr Yankam Joel

## Organisation
GroupeGeniusRepetitions

## Contenu
- Page d'accueil avec la liste des modules
- Page QCM avec validation, correction et explications
- Pagination des questions et navigation vers le module suivant

## Ajouter un module
Ouvre `data.js` et ajoute un nouvel objet dans le tableau `window.GENIUS_MODULES`.
Le numéro du module (`id`) s'affiche automatiquement.
